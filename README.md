# Cores – Manipulação de DOM e Geração Aleatória

Projeto simples em JavaScript desenvolvido como exercício prático para treinamento de manipulação de DOM e geração de cores aleatórias.

## Contexto

O projeto foi originalmente proposto como desafio no curso Full-Stack Turbo (TipsCode).  
Após o aprendizado inicial, foi refeito integralmente do zero como forma de consolidação dos conceitos.

Esta versão corresponde à reimplementação realizada no final de 2025.

Embora simples, este foi um dos primeiros projetos em JavaScript e marcou o início da aplicação prática de manipulação de elementos na interface.

## Objetivo

Praticar:

- Seleção e manipulação de elementos no DOM
- Registro de eventos (`addEventListener`)
- Uso de `Math.random()` para geração de valores aleatórios
- Manipulação dinâmica de estilos (`style.backgroundColor`)
- Estruturação básica de múltiplos arquivos JavaScript (modo `type="module"`)

## Funcionalidades

O projeto possui duas abordagens para geração de cores:

### 1. Modo Nome
- Seleciona aleatoriamente uma cor a partir de uma lista pré-definida
- Atualiza o fundo da página
- Exibe o nome da cor correspondente

### 2. Modo Hexadecimal
- Gera dinamicamente um código hexadecimal aleatório
- Atualiza o fundo da página
- Exibe o valor hexadecimal gerado

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (DOM API)

## Observação

Projeto desenvolvido exclusivamente para fins de estudo e consolidação dos fundamentos de JavaScript.
